# Awesome XREAL [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of awesome XREAL (formerly Nreal) AR glasses projects, libraries, tools, and resources organized by use case.

XREAL (formerly known as Nreal) creates AR glasses that overlay digital content onto the real world. This list focuses on open-source projects, development tools, and community resources for XREAL devices including Air, Air 2, Air 2 Pro, Light, and other AR glasses.

## Official Resources

- [XREAL Developer Portal](https://developer.xreal.com/) - Official developer documentation and resources
- [XREAL SDK Documentation](https://docs.xreal.com/) - Comprehensive SDK documentation and guides
- [NRSDK (Legacy)](https://github.com/nreal-ai) - Previous SDK versions for Nreal devices

## Development Tools & SDKs

- [XREAL SDK](https://docs.xreal.com/) - Official SDK for Unity development with AR features (hand tracking, spatial anchors, depth mesh, image tracking)
- [ar-drivers-rs](https://github.com/badicsalex/ar-drivers-rs) - Rust driver for various AR glasses including XREAL

## Virtual Desktop & Display

- [Breezy Desktop](https://github.com/wheaney/breezy-desktop) - XR virtual workspace library for Linux (532⭐)
- [XRLinuxDriver](https://github.com/wheaney/XRLinuxDriver) - Linux service for interacting with XR devices including XREAL (344⭐)
- [nreal_linux_test](https://github.com/alexwilson1/nreal_linux_test) - POC for multiple screens on Linux with NReal Air
- [AirAPI_Windows](https://github.com/MSmithDev/AirAPI_Windows) - Windows API for XREAL Air glasses (134⭐)

## Head Tracking & Motion Control

- [PhoenixHeadTracker](https://github.com/iVideoGameBoss/PhoenixHeadTracker) - Head tracking program for XREAL Air with OpenTrack UDP support (216⭐)
- [VRto3D](https://github.com/oneup03/VRto3D) - OpenVR driver for rendering in SbS or TaB 3D (94⭐)

## Gaming & Entertainment

- [decky-XRGaming](https://github.com/wheaney/decky-XRGaming) - Decky plugin for 3DoF effects on Steam Deck (154⭐)
- [spidgets-3dof](https://github.com/3rl-io/spidgets-3dof) - Example app for spatial widgets with 3DoF

## AR Applications

- [XREAL Air 2 Jarvis AR Assistant](https://github.com/r2w34/xreal-air2-jarvis-ar-assistant) - Complete Unity AR application with AI assistant, navigation, and voice control
- [Augmented Reality Project for XREAL Air](https://github.com/GalEvaldas/Augmented-Reality-Project) - Video playback AR project for dental procedures
- [realtime-language-translator-unity-client](https://github.com/harowosegbe/realtime-language-translator-unity-client) - Real-time language translation for Unity/AR (53⭐)
- [stereopsis-anything](https://github.com/zjkhurry/stereopsis-anything) - Convert 2D content to stereoscopic images compatible with AR/VR glasses (42⭐)

## Drivers & System Integration

- [real-air](https://github.com/abls/real-air) - Real Nreal Air driver implementation
- [android-nreal](https://github.com/enricoros/android-nreal) - Nreal Android application driver

## Unity & Mixed Reality Toolkit

- [NRSDK-MRTK-Samples](https://github.com/nreal-ai/NRSDK-MRTK-Samples) - Official MRTK compatibility samples with hand tracking (57⭐)
- [MRTK-Profiles-for-NrealLight](https://github.com/TakashiYoshinaga/MRTK-Profiles-for-NrealLight) - MRTK profiles configured for NReal Light (50⭐)
- [MRTKExtensionForNrealLight](https://github.com/HoloLabInc/MRTKExtensionForNrealLight) - MRTK extension for Nreal Light
- [UnityChatSDK](https://github.com/ShanguUncle/UnityChatSDK) - Unity video chat SDK supporting multiple platforms including AR glasses (232⭐)
- [Conekton](https://github.com/MESON-inc/Conekton) - Connectivity toolkit for Unity AR/VR projects

## WebXR & Browser-based

- [xreal-webxr](https://github.com/jakedowns/xreal-webxr) - WebXR support for XREAL devices

## Hardware & Accessories

- [Bento](https://github.com/lunchbox-computer/bento) - Computer in a keyboard design with XREAL integration (385⭐)
- [Steam Brick](https://github.com/crastinator-pro/steam-brick) - Modifying Steam Deck into compact AR gaming device (159⭐)
